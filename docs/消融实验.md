# ADHD 分类器 特征消融实验 (Ablation Study)

## 1. 简介

**目的**: 量化不同特征子集对 ADHD 二分类性能的贡献，验证当前线上模型所用的 12 维特征是否合理，并回答三个问题:

1. 哪个特征子集在 LOOCV 下准确率最高?
2. 瞳孔 / 注视 / 行为 / 条件 四个特征家族各自能贡献多少?
3. 相比纯行为 7 特征，加入瞳孔与注视是否带来实质增益?

**数据**: `model/adhd_pupil_classifier/features.csv`，共 50 名受试者 (28 ADHD / 22 Control)，27 维特征。

**模型**: `RandomForestClassifier(n_estimators=300, random_state=42)`，前置 `StandardScaler`。

**评估**: Leave-One-Out Cross-Validation (LOOCV, n_splits=50)。**标准化在每一折训练集内部 fit**，仅对留出样本 transform，避免数据泄露。指标 precision / recall / F1 针对 ADHD 类 (label=1)。

## 3. 关键发现

### 3.1 Top 3 特征子集

1. **选定 12 特征 (selected) - 线上模型** — accuracy = 0.8400，F1(ADHD) = 0.8621 (12 维)。
2. **无注视 (all - gaze, 24)** — accuracy = 0.8200，F1(ADHD) = 0.8421 (24 维)。
3. **全 27 特征 (all)** — accuracy = 0.8000，F1(ADHD) = 0.8214 (27 维)。

### 3.2 特征家族贡献

- **行为特征** (7 维) 单独 accuracy = 0.7600，是最强的单一家族之一。这符合 ADHD 临床表现中反应时变异性与正确率异常的先验。
- **瞳孔特征** (11 维) 单独 accuracy = 0.6600，反映认知负荷下的唤醒调节。
- **条件特征** (6 维) 单独 accuracy = 0.5600，衡量被试在不同负荷 / 干扰下的表现差异。
- **注视特征** (3 维) 单独 accuracy = 0.6600，维度最低、最弱，但作为轻量补充仍可能有用。

### 3.3 去除某一家族的影响

- 去掉瞳孔: accuracy 从 0.8000 → 0.7800 (Δ = +0.0200)。
- 去掉注视: accuracy 从 0.8000 → 0.8200 (Δ = -0.0200)。
  → 瞳孔特征对全量模型是**正贡献**，证明其必要性。

### 3.4 选定 12 特征是否合理?

- **选定 12 特征** accuracy = 0.8400，F1(ADHD) = 0.8621。
- vs **纯行为 7 特征** (0.7600): Δ = +0.0800
- vs **全 27 特征** (0.8000): Δ = +0.0400
- **结论**: 选定 12 特征**同时优于**纯行为基线与全量 27 维，说明所选特征既注入了行为之外的信息 (瞳孔/注视/条件)，又剔除了冗余噪声，是一个合理的工程折中。
